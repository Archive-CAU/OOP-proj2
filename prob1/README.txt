# Prob1 README

## Settings
OS: Windows 11
VS: Microsoft Visual Studio Community 2022 (64-bit) - 17.1.6

## Implementation using exe
1. go to "x64" directory
2. go to "Release" directory
3. click "prob1.exe"
4. input any key to exit program

## Implementation in VS
1. open "prob1.sln"
2. push "f5' (this solution automatically execute prob1)
	1) or right clik on "prob1" in solution explorer
	2) click "디버그" and click "새 인스턴스 시작" 
3. input any key to exit program
