# Prob2 README

## Settings
OS: Windows 11
VS: Microsoft Visual Studio Community 2022 (64-bit) - 17.1.6

## Implementation using exe
1. go to "x64" directory from "proj2" not "prob2"
2. go to "Release" directory
3. click "prob2.exe"
4. use that program (for example input "1234 + 5678")
5. input "0" to exit program

## Implementation in VS
1. open "prob1.sln"
2. don't push "f5', this solution automatically execute prob1
	0) if you want to compile and run this program, 
	1) right clik on "prob2" in solution explorer
	2) click "디버그" and click "새 인스턴스 시작" 
4. use that program (for example input "1234 + 5678")
5. input "0" to exit program

** 추가 한글번역 설명 **
f5를 눌러서 실행할 시, 이 sln은 prob1에 있는 main을 실행하게 됩니다.
